# Claude Code Guidelines

## Commits

- Do **not** add `Co-Authored-By` trailers to commit messages
- The user is directing all work — Claude is a tool, not a contributor
- Use conventional commit format: `type(scope): description`

## General

- This is a personal dotfiles repo managed with chezmoi
- Prefer small, focused commits grouped by logical change
- Always push after committing unless told otherwise
